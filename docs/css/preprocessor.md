### 1.2 概览

传统的 CSS 写法

```css
body {
  color: #fff;
}
```

在 Stylus 中，可以使用风格类似于 [pug](https://pugjs.org/api/getting-started.html) 的纯缩进语法改写为：

```stylus
body
  color white
```

::: tip
Stylus 也是兼容纯粹的 CSS 语法的。
:::
