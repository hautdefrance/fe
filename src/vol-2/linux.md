# _Linux_

# Command

> Sort alphabetically.

- **_ds_**
  - **du -lh**：Check the device's usage

- **_du_**
  - **du -h \[filename\]**：Readable file size
  - **du -a**：Show file

- **_ls_**
  - **ls -a**：Show hidden files
  - **ls -h**：Readable file size
  - **ls -l**：Show detailed info, contain file's permissions, owner, file size etc.
