# 清除浮动

  • 为什么要清楚浮动？

当一个内层元素是浮动的时候，如果没有关闭浮动，其父元素也就不会再包含这个浮动的内层元素，因为此时浮动元素已经脱离了文档流——这将导致外层元素不能被撑开，从而可能导致视图塌陷。

# 不清除浮动演示

  • Output:





  • Source Code:
<template>
  <div class="items">
    <div class="item"></div>
    <div class="item"></div>
    <div class="item"></div>
  </div>
</template>

<style scoped>
  .items {
    background-color: #f1f1f1;
    border: 3px solid #DA5961;
    width: 30%;
    margin: 0 auto;
  }

  .item {
    float: left;
    width: 30%;
    padding-bottom: 30%;
    margin-right: 2%;
    background-color: #3eaf7c;
  }
</style>

# 清除浮动演示

  • Output:


  • Source Code:
<template>
  <div class="items clearfix">
    <div class="item"></div>
    <div class="item"></div>
    <div class="item"></div>
  </div>
</template>

<style lang="stylus" scoped>
  .items {
    background-color: #f1f1f1;
    border: 3px solid #DA5961;
    width: 30%;
    margin: 0 auto;
    .item {
      float: left;
      width: 30%;
      padding-bottom: 30%;
      margin-right: 2%;
      background-color: #3eaf7c;
    }
  }

  .clearfix {
    zoom: 1;
    &::after {
      content: ".";
      display: block;
      height: 0;
      width: 0;
      clear: both;
      visibility: hidden
    }
  }
</style>

# 使用CSS画各种形状

# 自适应正方形

三种方式：

  1. 用css3的新单位 vh、vw
  2. padding-bottom = width = 某百分比
    • 缺点1：不能有文字填充
    • 缺点2：由于该块状元素没有高度，因为设置max-height失效
  3. 利用伪类来撑开父容器，父容器设置为某百分比后，添加一个伪类设置margin-top: 100%。注意，由于margin-collapse(外边距塌陷)的问题，
    • 缺点：内部仍然无法包含文字。但是可以设置max-height
  • Input:
<template>
  <div class="css-square"></div>
</template>

<style scoped>
  .css-square {
    width: 20%;
    padding-bottom: 20%;
    background-color: #3eaf7c;
  }
</style>
  • Output:

Tip: 试试缩小浏览器视口试试看~

# 多边框

# 方案1

  • 嵌套 box，最直观。
<template>
  <div class="multiple-border-1 box"></div>
</template>

<style lang="stylus" scoped>
  .box {
    width: 120px;
    height: 120px;
    padding: 5px 0 0 5px;
    box-sizing: border-box;
    background-color: green;
  }

  .box::after {
    content: "";
    display: block;
    width: 100px;
    height: 100px;
    border: 5px solid red;
    background-color: black;
  }
</style>
